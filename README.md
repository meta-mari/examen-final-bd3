# Examen Final BD3 - Farmacias

Proyecto académico para el diseño, transformación, implementación y documentación de una base de datos de farmacias.

## Contexto

El trabajo documenta el diseño conceptual, lógico y físico de una base de datos para farmacias, usando:

- Draw.io / diagrams.net para el modelo conceptual Chen.
- ER/Studio 2003 para el modelo lógico Crow's Foot.
- Oracle Database 19c en Docker para la implementación física.
- DBeaver para ejecución, validación y capturas de evidencia.
- LuaLaTeX + Biber para la compilación del informe final en formato APA 7.
